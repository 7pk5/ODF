# How to Run the ODF Project

This guide demonstrates how to run the project starting from activating the virtual environment.

## Prerequisites

- Python installed
- Command Prompt or PowerShell

## Steps

### 1. Navigate to the Project Directory

Open your terminal and navigate to the project folder:

```powershell
cd d:\FInalYear-Krunal\ODF
```

### 2. Activate the Virtual Environment

Activate the `odf_env` virtual environment:

**Command (PowerShell/CMD):**

```powershell
.\odf_env\Scripts\activate
```

You should see `(odf_env)` appear at the start of your command line.

### 3. Install/Update Dependencies (If needed)

Ensure all required packages are installed:

```powershell
pip install -r requirements.txt
```

### 4. Run the Application

Start the main application:

```powershell
python main.py
```
